1. Introduction

2. Présentation des données

Dans cette étude, les deux jeux de données suivants sont utilisés :

  • Données de consommation des auxiliaires agrégées à la mission, référencées dans la suite par effectifs.
  • Données sur le suivi temporel des mesures réalisées par les capteurs des climatisations, référencées dans la suite par hvac_all.

Ces deux jeux de données se limitent aux missions entre les deux terminus de la ligne "ANGERS - ROSERAIE" et "AVRILLE - ARDENNE" dans les deux sens.

D'un point de vue pratique, les données précédentes sont extraites via des requêtes SQL exécutées dans l'environnement SandBox de la plateforme Snowflake déployée chez RATP Dev.
